# 🗺️ My Live Map

An interactive Live Map web application built using HTML, CSS, JavaScript, and Leaflet.js. This app displays the user's real-time location and allows searching for places using OpenStreetMap’s Nominatim API with dynamic map updates.

---

## 🚀 Features

- 📍 Real-time user location using Geolocation API  
- 🗺️ Interactive map powered by Leaflet.js  
- 🔍 Search any place or address  
- 📌 Dynamic markers for searched locations  
- 🌐 Uses OpenStreetMap tiles (free & open-source)  
- ⚡ Fast and lightweight (no backend required)  
- 📱 Responsive UI with clean layout  

---

## 🛠️ Tech Stack

- HTML  
- CSS  
- JavaScript (Vanilla JS)  
- Leaflet.js  
- OpenStreetMap API (Nominatim)

## 🎯 Usage

1. Open the app  
2. Allow location access  
3. Your current location will be shown on the map  
4. Enter any place or address in the search box  
5. Click "Search" to navigate to that location
